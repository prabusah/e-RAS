How to

Abstract

Underwriters currently analyze the risk using Customer provided data, Medical reports etc. "e-RAS" system provides additional risk analyzing data from Social Media. This system not a replacement of existing methods, instead this is complimentry system.

Outcome

In addition to traditional inputs such as Customer provided data and medical info etc. The underwriter can make use of Social media in analyzing the risk.

How to run in local: (Move to project folder then run below commands)
  • npm install
  • gulp
  • node server.js
  • Launch : http://localhost:8080/home/index
